# `getCrypto`

## `monthly` `weekly` & `daily`

<Tip>
	`MONTHLY` , `WEEKLY` & `DAILY` all have the same parameters and the same
	response fields.
</Tip>
<Info>
	This API returns intraday time series (timestamp, open, high, low, close,
	volume) of the cryptocurrency specified, updated realtime. **Alpha Vantage
	Function**: `DIGITAL_CURRENCY_WEEKLY`
	[¶](https://www.alphavantage.co/documentation/#currency-monthly)
</Info>
### Parameters
<ParamField
	query="symbol"
	type="String!"
	required
>
	The cryptocurrency symbol. Example: `BTC` for Bitcoin.
</ParamField>
<ParamField
	query="market"
	type="String!"
	default="CNY"
>
	The market to compare the currency type against. Defaults to `CNY`
</ParamField>

### Response

    	type="DigitalCurrencyInterface!"
    >

<Expandable>

			type="DigitalCurrencyMetadata!"

			<Expandable>
				<ResponseField
					name="information"
					type="String!"
				/>
				<ResponseField
					name="digitalCurrencyCode"
					type="String!"
				/>
				<ResponseField
					name="digitalCurrencyName"
					type="String!"
				/>
				<ResponseField
					name="marketCode"
					type="String!"
				/>
				<ResponseField
					name="marketName"

## `intraday`

<Info>
	This API returns intraday time series (timestamp, open, high, low, close,
	volume) of the cryptocurrency specified, updated realtime. **Alpha Vantage
	Function**: `CRYPTO_INTRADAY`
	[¶](https://www.alphavantage.co/documentation/#crypto-intraday)
</Info>
### Parameters
<ParamField
	query="symbol"
	type="String!"
	required
>
	The cryptocurrency symbol. Example: `BTC` for Bitcoin.
</ParamField>
<ParamField
	query="interval"
	type="String!"
	default="5min"
>
	Time interval between two consecutive data points in the time series. Example:
	`5min` for 5 minutes. Accepted Intervals: `1min`, `5min`, `15min`, `30min`,
	`60min`
</ParamField>

### Response

<ResponseField name="Response" type="CurrencyExchangeRateType!">
  <Expandable title="properties">
    <ResponseField name="metadata" type="CryptoMetadataType!">
      <Expandable title="properties">
        <ResponseField name="digitalCurrencyCode" type="String!" />
        <ResponseField name="digitalCurrencyName" type="String!" />
        <ResponseField name="information" type="String!" />
        <ResponseField name="interval" type="String!" />
        <ResponseField name="lastRefreshed" type="String!" />
        <ResponseField name="marketCode" type="String!" />
        <ResponseField name="marketName" type="String!" />
        <ResponseField name="outputSize" type="String!" />
        <ResponseField name="timeZone" type="String!" />
      </Expandable>
    </ResponseField>
    <ResponseField name="series" type="TimeSeriesInterface!">
      <Expandable>
        <ResponseField name="open" type="String!" />
        <ResponseField name="date" type="String!" />
        <ResponseField name="high" type="String!" />
        <ResponseField name="low" type="String!" />
        <ResponseField name="close" type="String!" />
        <ResponseField name="volume" type="String!" />
      </Expandable>
    </ResponseField>

  </Expandable>
</ResponseField>
