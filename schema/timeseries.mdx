# `getTimeSeries`

## `weekly`

<Info>
	This API returns time series (open, high, low, close, volume) of the stock
	specified. **Alpha Vantage Function**: `TIME_SERIES_WEEKLY`
	[¶](https://www.alphavantage.co/documentation/#time-series)
</Info>

### Parameters

<ParamField
	query="symbol"
	type="String!"
	required
>
	The ticker symbol of the security
</ParamField>

### Response

<ResponseField
	name="metadata"
	type="TimeSeriesMetadata!"
>
	<Expandable>
		<ResponseField
			name="information"
			type="String!"
		/>
		<ResponseField
			name="symbol"
			type="String!"
		/>
		<ResponseField
			name="lastRefreshed"
			type="String!"
		/>
		<ResponseField
			name="outputSize"
			type="String!"
		/>
		<ResponseField
			name="timeZone"
			type="String!"
		/>
	</Expandable>
</ResponseField>
<ResponseField
	name="data"
	type="TimeSeriesData!"
>
	<Expandable>
		<ResponseField
			name="open"
			type="Float!"
		/>
		<ResponseField
			name="close"
			type="Float!"
		/>
		<ResponseField
			name="high"
			type="Float!"
		/>
		<ResponseField
			name="low"
			type="Float!"
		/>
		<ResponseField
			name="volume"
			type="Float!"

		<ResponseField
			name="date"
			type="String!"
		/>
	</Expandable>
</ResponseField>

## `intraday`

<Info>
	This API returns time series (open, high, low, close, volume) of the stock
	specified. **Alpha Vantage Function**: `TIME_SERIES_INTRADAY`
	[¶](https://www.alphavantage.co/documentation/#time-series)
</Info>

### Parameters

<ParamField
	query="symbol"
	type="String!"
	required
>
	The ticker symbol of the security
</ParamField>

<ParamField
	query="outputsize"
	type="String!"
	default="compact"
>
	The size of the output data. By default, the API returns compact.
</ParamField>

<ParamField
	query="interval"
	type="String!"
	default="15min"
>
	Time interval between two consecutive data points in the time series. Example:
	`5min` for 5 minutes. Accepted Intervals: `1min`, `5min`, `15min`, `30min`,
	`60min`
</ParamField>
