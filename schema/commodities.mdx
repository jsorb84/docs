# `getCommodities`

<Tip>
	All global commodities have the same parameters and the same return type ,
	this means they can be used interchangably.

    **AlphaVantage Documentation**: [here](https://www.alphavantage.co/documentation/)

</Tip>

### Parameters

<ParamField
	query="interval"
	default="monthly"
>
	Possible intervals: `monthly`, `quarterly`, and `annual`
</ParamField>

### Response

<Expandable>
	<ResponseField
		name="name"
		type="String!"
	/>
	<ResponseField
		name="interval"
		type="monthly | quarterly | annual"
		default="monthly"
	/>
	<ResponseField
		name="unit"
		type="String!"
	/>
	<ResponseField
		name="data"
		type="[CommodotiesDataInterface]!"
	>
		<Expandable>
			<ResponseField
				name="date"
				type="String!"
			/>
			<ResponseField
				name="value"
				type="String!"
			/>
		</Expandable>
	</ResponseField>
</Expandable>
